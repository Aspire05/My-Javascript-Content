Complex Data Types;

1. Object

Represents a collection of key-value pairs. Objects can be created with the Object constructor or using object literal notation {}.

2. Array

An array is a collection of values, and each value is identified by an index or a key.


Difference between Primitive Data Types and Complex Data Types

Primitive data types are basic and individual values, whereas complex data types are composed of primitive data types and can hold more complex structures, often with associated behaviors or methods. The distinction is essential for understanding how different types of data are stored and manipulated in a programming language.
